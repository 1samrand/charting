# charting

گزارش تحلیلی برای تبدیل داده‌های اکسل مالی به نمودارهای مدیریتی.

## خروجی نمونه

- `reports/sample-financial-report-fa.html`: گزارش مدیریتی راست‌به‌چپ با KPI، نمودارهای درآمد/هزینه/حاشیه سود و جدول شاخص‌های فصلی.
- `data/sample-financial-data.json`: نسخه تمیزشده داده نمونه برای استفاده در مرحله بعدی ساخت اپلیکیشن.

برای مشاهده گزارش، فایل HTML را مستقیما در مرورگر باز کنید.

## دامنه تحلیل فعلی

- گزارش فقط ردیف‌های داده‌شده تا «سود عملیات در حال تداوم قبل از مالیات» را پوشش می‌دهد.
- ردیف‌های بعد از «سود عملیات در حال تداوم قبل از مالیات» در این نسخه لحاظ نشده‌اند.
- درصدهای کنار هر ردیف به عنوان نسبت آن ردیف به درآمد عملیاتی همان فصل در نظر گرفته شده‌اند.
