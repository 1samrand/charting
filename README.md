# charting

گزارش تحلیلی برای تبدیل داده‌های اکسل مالی به نمودارهای مدیریتی.

## خروجی گزارش

- `reports/financial-report-fa.html`: گزارش مدیریتی راست‌به‌چپ با KPI، نمودارهای درآمد، هزینه، حاشیه سود و جدول شاخص‌های فصلی.
- `data/financial-report-data.json`: داده ساختاریافته گزارش برای استفاده در اپلیکیشن.

برای مشاهده گزارش، فایل HTML را مستقیما در مرورگر باز کنید.

## مبنای تحلیل

- درصدهای کنار هر ردیف به عنوان نسبت آن ردیف به درآمد عملیاتی همان فصل در نظر گرفته شده‌اند.
- عنوان «سود عملیات قبل از مالیات» معادل ردیف پایانی داده ارائه‌شده در فایل اکسل است.
