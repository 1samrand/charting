# charting

نمونه اولیه برای تبدیل فایل‌های اکسل مالی به گزارش و نمودار کاربردی.

## خروجی نمونه

- `reports/sample-financial-report-fa.html`: گزارش مدیریتی راست‌به‌چپ با KPI، نمودارهای درآمد/سود، حاشیه سود، سهم فصل‌ها و کنترل کیفیت داده.
- `data/sample-financial-data.json`: نسخه تمیزشده داده نمونه برای استفاده در مرحله بعدی ساخت اپلیکیشن.

برای مشاهده گزارش، فایل HTML را مستقیما در مرورگر باز کنید.

## نکته‌های داده

- عدد سود خالص فصل‌ها با ستون کل اختلاف 2 واحدی دارد.
- ردیف «هزینه غیر قابل پیش‌بینی» در بهار مثل هزینه کسر شده، اما در فصل‌های بعدی مثل درآمد/افزایش سود عمل کرده است؛ پیش از خودکارسازی باید علامت این ردیف در فایل اصلی مشخص شود.
- ردیف مالیات سال جاری فقط در ستون کل مقدار دارد و سیاست تخصیص فصلی آن هنوز مشخص نیست.
